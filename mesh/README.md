# 网格文件目录说明

本目录用于存放测试用的网格文件。

## 网格文件格式

- 文件格式：Gmsh格式（.msh）
- 命名规范：小写+下划线，如 `motor2d_simple.msh`
- 编码格式：UTF-8

## 测试网格说明

### 当前可用的测试网格

- **motor2d_simple.msh** - 2D简易电机网格（待创建）
- **airgap3d.msh** - 3D气隙网格（待创建）

## 网格生成工具

推荐使用以下工具生成网格：

1. **Gmsh** - 开源网格生成器
   - 官网：https://gmsh.info/
   - 支持2D/3D网格生成
   - 支持多种网格格式

2. **Salome** - 开源CAE平台
   - 官网：https://www.salome-platform.org/
   - 提供完整的网格生成流程

## 网格质量要求

- 单元质量：雅可比矩阵行列式 > 0
- 网格密度：关键区域适当加密
- 边界层：电磁场变化剧烈区域需加密
- 界面处理：子域界面网格需匹配
