# ANSYS Maxwell 静磁场完整功能清单（工业级）
Maxwell静磁场是**所有磁场类求解器的基础**，与静电场、直流电流场共享90%以上的底层代码（网格、DOF管理、矩阵装配、线性求解器），核心差异仅在物理模型、材料参数、激励源和后处理计算。它是电机、变压器、电磁铁、永磁器件等工业设备设计的核心工具。

## 一、核心求解能力
- **维度支持**：完整支持**二维平面(XY)**、**二维轴对称(RZ)**和**三维**模型
- **控制方程**：
  - 2D静磁场：磁矢位z分量公式 $\nabla\cdot(\frac{1}{\mu}\nabla A_z)=-J_z$
  - 3D静磁场：磁矢位矢量公式 $\nabla\times(\frac{1}{\mu}\nabla\times\mathbf{A})=\mathbf{J}$
  - 无电流区：可使用磁标量位公式（计算更快）
- **未知量**：2D为节点标量磁矢位$A_z$；3D为棱边矢量磁矢位$\mathbf{A}$（Nédélec单元）
- **求解精度**：基于能量误差的自适应求解，默认能量误差<1%，可设置到0.1%
- **数值方法**：支持1阶和2阶拉格朗日单元（2D）、1阶和2阶棱边单元（3D）
- **非线性支持**：完整支持铁磁材料的非线性B-H曲线，采用牛顿-拉夫逊迭代法求解

## 二、支持的边界条件
| 边界条件类型 | 物理意义 | 典型应用 |
|-------------|---------|---------|
| **磁壁边界(Magnetic Wall)** | 磁场切向分量为零，磁力线垂直于边界 | 模型对称面、磁屏蔽表面 |
| **电壁边界(Electric Wall)** | 磁场法向分量为零，磁力线平行于边界 | 理想导体表面、模型对称面 |
| **气球边界(Balloon)** | 模拟无限远边界，磁场衰减到零 | 永磁体、电磁铁、电机的远场分析 |
| **对称边界(Symmetry)** | 等同于磁壁边界 | 利用模型对称性减少计算量 |
| **反对称边界(Anti-Symmetry)** | 等同于电壁边界 | 利用模型对称性减少计算量 |
| **周期性边界(Periodic)** | 主从边界，磁场分布重复 | 电机定子/转子、阵列式磁体、变压器铁芯 |
| **主从边界(Master-Slave)** | 强制两个边界的磁场分布相同 | 旋转电机的滑动边界（静磁场中用于静态转矩计算） |
| **默认边界** | 3D中默认是磁壁边界；2D中默认是电壁边界 | 模型外边界 |

## 三、支持的激励源
1. **电流源激励**（最常用）：
   - **电流密度激励**：指定导体内部的电流密度（A/m²）
   - **电流激励**：指定导体截面的总电流（A），自动计算电流密度
   - **电压激励**：指定导体两端的电压，自动计算电流和电流密度
   - **绕组激励**：支持集中绕组和分布绕组，可指定匝数、并联支路数、相序

2. **永磁体激励**：
   - 支持剩磁$B_r$和矫顽力$H_c$两种定义方式
   - 支持任意方向的磁化方向
   - 支持温度相关的永磁体参数
   - 支持退磁曲线（用于分析永磁体的退磁风险）

3. **外磁场激励**：
   - 均匀外磁场
   - 梯度外磁场
   - 用于分析磁体在外部磁场中的受力和运动

4. **悬浮导体/磁体激励**：
   - 总电流为0的悬浮导体
   - 总磁通量为0的悬浮磁体
   - 用于分析感应带电和感应磁化现象

## 四、支持的材料模型
1. **线性磁性材料**：
   - 相对磁导率$\mu_r$为常数
   - 适用于空气、铜、铝等非铁磁材料

2. **非线性铁磁材料**：
   - 支持B-H曲线和H-B曲线的双向插值
   - 支持各向同性和各向异性铁磁材料
   - 支持温度相关的B-H曲线
   - 支持磁滞回线（预留接口，用于瞬态场）

3. **永磁体材料**：
   - 支持剩磁$B_r$和矫顽力$H_c$
   - 支持退磁曲线
   - 支持温度相关的永磁体参数

4. **导体材料**：
   - 电导率$\sigma$（用于计算欧姆损耗）
   - 支持温度相关的电导率

5. **绝缘体材料**：
   - 电导率为0，不导电
   - 相对磁导率$\mu_r=1$

## 五、核心后处理计算（工业价值最高）
### 1. 基础场量计算
- 磁矢位分布$\mathbf{A}$
- 磁感应强度分布$\mathbf{B}=\nabla\times\mathbf{A}$
- 磁场强度分布$\mathbf{H}=\frac{\mathbf{B}}{\mu}$
- 电流密度分布$\mathbf{J}=\nabla\times\mathbf{H}$
- 磁能密度分布$w_m=\frac{1}{2}\mathbf{B}\cdot\mathbf{H}$
- 磁力密度分布

### 2. 核心积分量计算（必须优先实现）
- **电感矩阵计算**：自动计算任意多个绕组之间的自电感和互电感
  - 自电感$L_{ii}=\frac{\Psi_i}{I_i}$
  - 互电感$L_{ij}=\frac{\Psi_i}{I_j}$
  - 漏电感和主电感的分离
- **磁通量计算**：任意闭合面的总磁通量
- **磁链计算**：任意绕组的总磁链
- **磁场能量计算**：总磁场能量$W_m=\frac{1}{2}\int_V\mathbf{B}\cdot\mathbf{H}dV$
- **电磁力计算**：
  - 麦克斯韦应力张量法（速度快，适用于整体受力）
  - 虚功原理法（精度高，适用于局部受力）
  - 洛伦兹力法（适用于导体受力）
  - 支持整体受力和局部受力计算
- **电磁转矩计算**：
  - 麦克斯韦应力张量法
  - 虚功原理法
  - 绕任意轴的转矩计算
  - 支持转矩-角度曲线自动生成
- **欧姆损耗计算**：导体中的焦耳热损耗$P=\int_V\mathbf{J}\cdot\mathbf{E}dV$
- **磁阻计算**：任意磁路的磁阻$R_m=\frac{F}{\Phi}$

# 静磁场求解的10大核心关键技术（工业级实现）
静磁场是所有磁场类求解器的基础，也是工业应用中最复杂、最容易出问题的部分。与静电场、直流电流场相比，静磁场求解面临**矢量场非唯一性、铁磁材料强非线性、3D伪解、力/转矩计算精度**四大核心挑战。以下是工业级求解器必须掌握的关键技术，完全对标ANSYS Maxwell的底层实现：

---

## 一、数学建模与公式选择技术（最基础也最关键）
### 1. 磁矢位法（A法）：主流工业实现
- **2D静磁场**：使用**标量磁矢位A_z**公式
  $$\nabla\cdot\left(\frac{1}{\mu}\nabla A_z\right) = -J_z$$
  未知量为节点标量A_z，使用拉格朗日单元，计算简单、精度高
- **3D静磁场**：使用**矢量磁矢位A**公式
  $$\nabla\times\left(\frac{1}{\mu}\nabla\times\mathbf{A}\right) = \mathbf{J}$$
  未知量为棱边矢量A，必须使用Nédélec棱边单元，避免伪解

### 2. 库仑规范处理：解决矢量位非唯一性
矢量磁矢位A存在无穷多解（A' = A + ∇φ），必须施加库仑规范∇·A=0才能得到唯一解。
- **工业实现方法**：Nédélec棱边单元天然满足离散库仑规范，无需额外处理（这也是必须使用棱边单元的核心原因）
- **常见错误**：使用节点单元求解3D矢量磁矢位，会产生大量非物理的"伪解"

### 3. 标量磁位法（φ法）：无电流区加速技术
在没有电流的区域（如空气、铁磁材料、永磁体），可以使用更简单的标量磁位φ公式：
$$\nabla\cdot(\mu\nabla\phi) = 0$$
- **优势**：每个节点只有1个自由度，计算速度比矢量位法快3-5倍
- **工业实现**：采用**标量位-矢量位混合方法**
  - 电流区域：使用矢量磁矢位A
  - 无电流区域：使用标量磁位φ
  - 交界面：施加连续性条件
- **Maxwell实现**：3D静磁场默认开启混合方法，可将大型模型的求解时间减少50%以上

---

## 二、单元技术：3D静磁场的核心
### 1. Nédélec棱边单元技术：解决3D伪解问题
这是3D静磁场求解器与其他求解器最大的区别，也是最容易出错的地方。
- **核心特点**：
  - 自由度分配在**棱边**上，而不是节点上
  - 每个棱边1个自由度，代表沿棱边的切向分量
  - 1阶Nédélec四面体单元：6个自由度/单元
  - 2阶Nédélec四面体单元：20个自由度/单元
- **为什么能避免伪解**：
  - 棱边单元的旋度空间是无散的，天然满足库仑规范
  - 不会产生非物理的"梯度场"伪解
- **实现难点**：
  - 棱边的全局编号与方向一致性
  - 形函数的计算与雅可比变换
  - 边界条件的施加（只能施加切向分量）

### 2. 轴对称模型特殊处理
- **坐标系转换**：将三维柱坐标系(r,θ,z)转换为二维平面坐标系(r,z)
- **控制方程修改**：
  $$\frac{\partial}{\partial r}\left(\frac{r}{\mu}\frac{\partial A_\theta}{\partial r}\right) + \frac{\partial}{\partial z}\left(\frac{r}{\mu}\frac{\partial A_\theta}{\partial z}\right) = -r J_\theta$$
- **积分权重调整**：所有体积分都要乘以r因子
- **后处理转换**：将二维结果转换为三维物理量

## 三、非线性求解技术：铁磁材料处理
铁磁材料的B-H曲线是非线性的，这是静磁场求解最耗时的部分。
### 1. 牛顿-拉夫逊迭代法：工业标准实现
- **迭代公式**：
  $$\mathbf{K}(\mathbf{A}^k)\Delta\mathbf{A}^{k+1} = \mathbf{F} - \mathbf{R}(\mathbf{A}^k)$$
  $$\mathbf{A}^{k+1} = \mathbf{A}^k + \Delta\mathbf{A}^{k+1}$$
- **核心步骤**：
  1. 初始化磁导率μ
  2. 组装刚度矩阵K和残差向量R
  3. 求解线性方程组得到ΔA
  4. 更新A并重新计算μ
  5. 检查收敛性，不收敛则返回步骤2
- **收敛准则**：
  - 残差范数：$\|\mathbf{R}\| < \epsilon_1$
  - 解的增量范数：$\|\Delta\mathbf{A}\| < \epsilon_2$
  - 能量误差：$\Delta W_m / W_m < \epsilon_3$（Maxwell默认使用）

### 2. 非线性加速技术
- **初始解预测**：使用上一个载荷步或上一次迭代的解作为初始解
- **线性搜索**：当迭代发散时，减小步长
- **材料插值优化**：使用三次样条插值B-H曲线，提高导数计算精度
- **欠松弛技术**：对磁导率进行欠松弛，提高迭代稳定性

### 3. 永磁体非线性处理
- **退磁曲线表示**：使用B-H曲线的第二象限部分
- **工作点计算**：自动计算永磁体的工作点
- **退磁风险评估**：当工作点低于膝点时，发出警告

## 四、边界条件处理技术
### 1. 基本边界条件
- **电壁边界（Dirichlet）**：A_t=0（磁场法向分量为零）
- **磁壁边界（Neumann）**：∂A/∂n=0（磁场切向分量为零）
- **气球边界**：模拟无限远边界，使用渐近展开近似

### 2. 周期性边界条件
- **平移周期性**：用于阵列式结构
- **旋转周期性**：用于电机定子/转子
- **实现方法**：建立主从边界的自由度映射关系，修改全局矩阵

### 3. 滑动边界条件（静态转矩计算）
- 用于计算不同转子角度下的静态转矩
- 实现方法：将模型分为静止区域和运动区域，在交界面施加约束方程
- **Maxwell实现**：自动生成不同角度的模型，批量求解并生成转矩-角度曲线

## 五、激励源处理技术
### 1. 电流源激励
- **电流密度激励**：直接指定导体内部的电流密度
- **总电流激励**：指定导体截面的总电流，自动计算电流密度
  $$J = \frac{I}{S}$$
- **绕组激励**：
  - 集中绕组：指定匝数和总电流
  - 分布绕组：指定绕组函数和总电流
  - 自动计算每个导体的电流密度

### 2. 电压源激励
- 需要迭代求解电流，因为电流与磁场有关
- **实现方法**：
  1. 假设一个初始电流
  2. 求解磁场得到绕组磁链Ψ
  3. 计算电压U = IR + dΨ/dt（直流时dΨ/dt=0）
  4. 调整电流直到电压等于指定值
  5. 重复迭代直到收敛

### 3. 永磁体激励
- **等效电流法**（工业标准）：将永磁体等效为面电流
  $$\mathbf{J}_m = \nabla\times\mathbf{M}$$
  其中M是磁化强度，M = B_r/μ_0 - H_c
- **优势**：与电流源激励的实现方式一致，代码复用性好
- **磁化方向处理**：支持任意方向的磁化，包括随几何变化的方向

## 六、后处理关键技术（工业价值核心）
### 1. 电感矩阵计算
- **能量法**（精度最高，工业标准）：
  $$L_{ij} = \frac{2W_m}{I_i I_j}$$
  其中W_m是总磁场能量
- **磁链法**：
  $$L_{ij} = \frac{\Psi_i}{I_j}$$
- **实现要点**：
  - 自电感和互电感的正确分离
  - 漏电感和主电感的计算
  - 非线性情况下的增量电感计算

### 2. 电磁力与转矩计算
这是工业应用中最看重的功能，也是最容易出错的地方。
- **麦克斯韦应力张量法**（速度快）：
  $$\mathbf{F} = \oint_S \left(\frac{1}{\mu}\mathbf{B}\mathbf{B}^T - \frac{1}{2\mu}B^2\mathbf{I}\right)\cdot d\mathbf{S}$$
  - 优点：计算速度快，只需要积分一个面
  - 缺点：对网格质量敏感，局部精度低
- **虚功原理法**（精度最高，Maxwell默认）：
  $$F_x = -\frac{\partial W_m}{\partial x}$$
  $$T_\theta = -\frac{\partial W_m}{\partial \theta}$$
  - 优点：精度高，对网格质量不敏感
  - 缺点：需要求解两次（位移前后）
- **洛伦兹力法**（适用于导体受力）：
  $$\mathbf{F} = \int_V \mathbf{J}\times\mathbf{B} dV$$

### 3. 磁场能量与损耗计算
- **总磁场能量**：
  $$W_m = \frac{1}{2}\int_V \mathbf{B}\cdot\mathbf{H} dV$$
- **欧姆损耗**：
  $$P_{cu} = \int_V \frac{J^2}{\sigma} dV$$
- **铁损计算**：静磁场中只考虑磁滞损耗，使用斯坦梅茨模型

## 七、性能优化技术（工业级求解器核心竞争力）
### 1. 矩阵装配优化
- **预计算**：预计算所有积分点的形函数、雅可比和权重
- **循环优化**：按"单元→积分点→棱边"的顺序循环，提高缓存命中率
- **多线程并行**：使用OpenMP对单元循环进行并行化
- **稀疏矩阵预分配**：提前预估非零元数量，避免动态扩容

### 2. 线性求解器调优
- **矩阵重排序**：使用AMD/COLAMD算法减少填充元
- **求解器选择**：
  - 小型模型：SuperLU
  - 大型模型：MUMPS或PARDISO
- **并行参数调优**：设置合适的线程数和内存块大小

### 3. 自适应网格技术
- 基于能量误差的网格加密
- 自动加密高磁场梯度区域
- 支持局部网格加密，避免全局加密

## 八、常见问题与解决方案
| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 3D求解出现非物理的高磁场 | 使用了节点单元，产生伪解 | 改用Nédélec棱边单元 |
| 非线性迭代不收敛 | 初始解不好，或B-H曲线不完整 | 使用更好的初始解，补充B-H曲线到高场 |
| 转矩计算结果波动大 | 麦克斯韦应力张量法对网格敏感 | 改用虚功原理法，加密交界面网格 |
| 电感计算结果与Maxwell偏差大 | 能量计算错误，或边界条件处理不当 | 检查能量积分，验证边界条件 |
| 大型模型内存不足 | 全局矩阵太大 | 使用标量位-矢量位混合方法，启用求解器的内存优化 |

## 九、技术实现优先级建议
1. **第一优先级**：
   - 2D静磁场线性求解（A_z法）
   - 电流源激励
   - 基础场量计算
   - 电感矩阵计算（能量法）
2. **第二优先级**：
   - 非线性铁磁材料处理（牛顿-拉夫逊法）
   - 永磁体激励
   - 电磁力与转矩计算（虚功原理法）
3. **第三优先级**：
   - 3D静磁场求解（Nédélec棱边单元）
   - 标量位-矢量位混合方法
   - 自适应网格技术
