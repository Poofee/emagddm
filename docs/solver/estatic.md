# ANSYS Maxwell 静电场完整功能清单（工业级）
Maxwell静电场是**所有电场类求解器的基础**，与直流电流场共享90%以上的底层代码（网格、DOF管理、矩阵装配、线性求解器），核心差异仅在物理模型、材料参数、激励源和后处理计算。以下是工业级应用中所有实用功能的详细拆解：

## 一、核心求解能力
- **维度支持**：完整支持**二维平面(XY)**、**二维轴对称(RZ)**和**三维**模型
- **控制方程**：泊松方程 $\nabla\cdot(\epsilon\nabla V)=-\rho$，无电荷区退化为拉普拉斯方程
- **未知量**：标量电位 $V$（所有节点自由度）
- **求解精度**：基于能量误差的自适应求解，默认能量误差<1%，可设置到0.1%
- **数值方法**：标量有限元法（拉格朗日单元），支持1阶和2阶单元

## 二、支持的边界条件
| 边界条件类型 | 物理意义 | 典型应用 |
|-------------|---------|---------|
| **电压边界(Voltage)** | 指定边/面的电位值（Dirichlet边界） | 电极、接地、高压端 |
| **电荷边界(Charge)** | 指定边/面的总电荷量 | 带电导体、静电感应 |
| **悬浮导体边界(Floating Conductor)** | 导体电位未知，总电荷已知（可为0） | 孤立金属件、感应带电体 |
| **绝缘边界(Insulating)** | 电位移矢量法向分量为零（默认边界） | 模型外边界、绝缘表面 |
| **对称边界(Symmetry)** | 电场法向分量为零 | 利用模型对称性减少计算量 |
| **反对称边界(Anti-Symmetry)** | 电场切向分量为零 | 利用模型对称性减少计算量 |
| **周期性边界(Periodic)** | 主从边界，电场分布重复 | 阵列结构、重复单元 |
| **气球边界(Balloon)** | 模拟无限远边界 | 高压设备、天线、接地系统 |

## 三、支持的激励源
1. **电压源激励**：最常用，可指定任意边/面的直流电压值
2. **电荷源激励**：
   - 体电荷密度（Volume Charge Density）
   - 面电荷密度（Surface Charge Density）
   - 线电荷（Line Charge）
   - 点电荷（Point Charge）
3. **悬浮导体激励**：
   - 总电荷为0的悬浮导体（感应带电）
   - 带指定净电荷的悬浮导体
4. **接地激励**：电位为0的特殊电压源

## 四、支持的材料模型
1. **介电材料**：
   - 各向同性介电常数 $\epsilon_r$
   - 各向异性介电常数张量
   - 温度相关介电常数
   - 损耗角正切 $\tan\delta$（用于介质损耗计算）
2. **理想导体**：内部电场为零，是等势体
3. **驻极体材料**：具有永久极化强度的电介质
4. **压电材料**：支持压电效应的耦合分析（预留接口）

## 五、核心后处理计算（工业价值最高）
### 1. 基础场量计算
- 电位分布 $V$
- 电场强度分布 $\mathbf{E}=-\nabla V$
- 电位移矢量分布 $\mathbf{D}=\epsilon\mathbf{E}$
- 电荷密度分布 $\rho=\nabla\cdot\mathbf{D}$
- 能量密度分布 $w_e=\frac{1}{2}\mathbf{D}\cdot\mathbf{E}$

### 2. 核心积分量计算（必须优先实现）
- **电容矩阵计算**：自动计算任意多个导体之间的自电容和互电容
  - 部分电容矩阵（Partial Capacitance Matrix）
  - 集总电容矩阵（Lumped Capacitance Matrix）
- **电场能量计算**：总电场能量 $W_e=\frac{1}{2}\int_V\mathbf{D}\cdot\mathbf{E}dV$
- **静电力计算**：
  - 麦克斯韦应力张量法（速度快）
  - 虚功原理法（精度高）
  - 支持整体受力和局部受力计算
- **静电力矩计算**：绕任意轴的力矩
- **电荷计算**：任意导体表面的总电荷量
- **电压差计算**：任意两点之间的电位差

### 3. 几何实体后处理
- 任意点的场量值
- 任意路径的场量分布曲线
- 任意面的电通量计算
- 任意体的总电场能量和总电荷
- 等值线和等值面计算
- 电场线（流线）计算

### 4. 可视化功能
- 电位等值线/等值面云图
- 电场强度云图
- 电位移矢量箭头图
- 电场线（流线）图
- 电荷密度云图
- 能量密度云图
- 动画功能（参数化扫描结果动画）

## 六、高级功能
### 1. 自适应网格技术
- 自动加密高电场梯度区域的网格
- 基于能量误差的自适应迭代
- 可设置最大迭代次数和目标误差
- 支持局部网格加密

### 2. 参数化分析与优化
- 几何参数扫参
- 材料参数扫参
- 激励参数扫参
- 与ANSYS OptiSLang集成，进行自动优化设计

### 3. 场计算器
- 支持任意场量的代数运算
- 支持矢量运算（点积、叉积、梯度、散度、旋度）
- 支持积分运算（线积分、面积分、体积分）
- 支持自定义后处理变量

### 4. 脚本自动化
- 支持Python和VBScript脚本
- 可实现建模、求解、后处理的全流程自动化
- 支持批量处理多个模型

## 七、多物理场耦合功能
### 1. 与直流电流场的耦合
- 开启"Include Insulator Field"选项
- 同时求解导体区域的电流场和绝缘体区域的静电场
- 适用于高压设备的绝缘分析和泄漏电流计算

### 2. 与结构场的耦合（静电-结构耦合）
- 单向耦合：静电场计算静电力，作为结构场的载荷
- 双向耦合：结构场计算变形，反馈给静电场更新几何
- 用于MEMS器件、静电执行器、静电吸盘的分析

### 3. 与热场的耦合
- 单向耦合：静电场计算介质损耗，作为热场的热源
- 双向耦合：热场计算温度分布，反馈给静电场更新介电常数
- 用于高压绝缘子、电容器的热分析

### 4. 与电路的耦合
- 支持与ANSYS Simplorer电路仿真软件的联合仿真
- 可以将静电场求解器生成的电容模型导入电路仿真
- 可以将电路仿真的电压波形作为静电场的激励源

## 八、典型应用场景
1. **电容器设计**：计算电容值、电场分布、能量密度
2. **高压绝缘设计**：绝缘子、套管、断路器的电场分析
3. **静电放电(ESD)分析**：评估静电放电风险
4. **MEMS器件**：静电执行器、加速度计、陀螺仪
5. **传感器**：电容式传感器、静电传感器
6. **印刷电路板(PCB)**：PCB走线之间的寄生电容分析
7. **电缆设计**：电缆的电场分布和绝缘性能分析
8. **静电除尘**：静电除尘器的电场分析
9. **静电喷涂**：静电喷涂设备的电场分析
10. **航天器静电防护**：航天器表面的静电电位分析

## 九、对你开发求解器的启示
### 1. 架构设计
完全复用之前设计的**通用标量电场求解器基类**：
```cpp
class ScalarElectricSolverBase : public PhysicsSolver {
public:
    void initialize() override;
    void assemble() override;
    SolverResult solve() override;

protected:
    virtual double getMaterialCoefficient(const Element& element) const = 0;
    virtual void applyExcitations() = 0;
    virtual void computeDerivedQuantities() = 0;

    std::shared_ptr<DOFManager> dof_manager_;
    std::shared_ptr<MatrixAssembler> matrix_assembler_;
    std::shared_ptr<LinearSolver> linear_solver_;
    std::vector<double> potential_;
};

// 静电场求解器
class ElectrostaticSolver : public ScalarElectricSolverBase {
protected:
    double getMaterialCoefficient(const Element& element) const override {
        return element.getMaterial().getPermittivity();
    }

    void applyExcitations() override; // 施加电压源和电荷源
    void computeDerivedQuantities() override; // 计算E、D、电容、静电力
};
```

### 2. 功能优先级
- **第一优先级**：基础场量计算、电压边界、电容矩阵计算、电场能量计算
- **第二优先级**：电荷源、悬浮导体、静电力计算、自适应网格
- **第三优先级**：耦合功能、参数化分析、优化设计

### 3. 核心注意事项
- 静电场的数值稳定性比电流场更好，但要注意处理高介电常数比的情况
- 电容矩阵计算是静电场最核心的功能，必须确保精度与Maxwell一致
- 悬浮导体的处理是静电场的难点，需要正确实现约束方程
