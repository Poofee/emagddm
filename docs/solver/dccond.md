# ANSYS Maxwell 电流场完整功能清单
Maxwell电流场包含**直流电流场(DC Conduction)**、**交流电流场(AC Conduction)**和**瞬态电流场(Transient Conduction)**三个独立求解器，共享90%以上的底层基础代码，核心差异仅在物理模型和后处理计算。以下是工业级应用中所有实用功能的详细拆解：

## 一、直流电流场(DC Conduction)功能（2D/3D全支持）
### 1. 核心求解能力
- 求解恒定电流在导体中的稳态分布
- 求解导体内部的电位分布和电场强度分布
- 自动计算电流密度分布（J=σE）
- 支持二维平面、二维轴对称和三维模型

### 2. 支持的边界条件
- **电压边界条件**：指定边/面的电位值（Dirichlet边界）
- **电流边界条件**：指定边/面的流入/流出电流值（Neumann边界）
- **绝缘边界条件**：电流法向分量为零（默认边界）
- **电阻边界条件**：仅2D支持，模拟薄层电阻
- **对称边界条件**：利用模型对称性减少计算量
- **周期性边界条件**：主从边界，模拟重复结构
- **气球边界条件**：模拟无限远边界

### 3. 支持的激励源
- **电压源激励**：直流电压，可指定任意边/面
- **电流源激励**：直流电流，可指定任意边/面
- **接地激励**：电位为0的特殊电压源

### 4. 支持的材料模型
- **导体材料**：需要定义电导率σ（S/m）
- **绝缘体材料**：电导率为0，默认不求解其内部电场
- **各向异性材料**：支持电导率张量
- **温度相关材料**：电导率随温度变化（用于电热耦合）

### 5. 核心后处理计算（工业价值最高）
- **电阻计算**：自动计算任意两个端子之间的直流电阻
- **电导计算**：电阻的倒数
- **电流计算**：任意截面的总电流（电流密度积分）
- **焦耳热计算**：任意区域的总焦耳热损耗（P=∫E·J dV）
- **功率密度分布**：单位体积的焦耳热功率
- **电压降计算**：任意两点之间的电压差
- **电场强度分布**：E=-∇V
- **电流密度分布**：J=σE
- **接地电阻计算**：用于接地系统分析
- **跨步电压计算**：用于高压接地安全分析
- **接触电阻计算**：通过薄层电阻边界条件模拟

### 6. 典型应用场景
- 连接器、母线排、PCB的电流分布分析
- 接地系统设计与接地电阻计算
- 电热器、加热元件的热功率计算
- 高压设备的跨步电压和接触电压分析
- 电池、燃料电池的电流分布分析

## 二、交流电流场(AC Conduction)功能（仅2D支持）
### 1. 核心求解能力
- 求解正弦稳态下的电流分布
- 同时考虑传导电流和位移电流
- 求解导体和介质中的电位分布
- 支持二维平面和二维轴对称模型

### 2. 控制方程
∇·[(σ + jωε)∇V] = 0
- σ：电导率（传导电流项）
- ε：介电常数（位移电流项）
- ω：角频率

### 3. 支持的边界条件和激励源
- 与直流电流场完全相同
- 所有激励源均为正弦量，可指定幅值和相位

### 4. 支持的材料模型
- 同时需要电导率σ和介电常数ε
- 支持各向异性材料
- 支持频率相关材料参数

### 5. 核心后处理计算
- **阻抗计算**：任意两个端子之间的复阻抗（Z=R+jX）
- **交流电阻计算**：R=P/I²
- **电容计算**：C=X/(ω)
- **传导损耗计算**：导体中的焦耳热损耗
- **介质损耗计算**：绝缘体中的介电损耗（P=ωε''|E|²/2）
- **总损耗计算**：传导损耗+介质损耗
- **电流密度分布**：复电流密度
- **电场强度分布**：复电场强度

### 6. 典型应用场景
- 电容器的介质损耗计算
- 绝缘子的泄漏电流和介质损耗分析
- 高频PCB的电流分布分析
- 电缆的绝缘性能分析

**重要注意**：Maxwell交流电流场**不考虑集肤效应和邻近效应**，仅适用于非良导体和低频应用。对于良导体的高频电流分布分析，应使用涡流场求解器。

## 三、瞬态电流场(Transient Conduction)功能（仅3D支持）
### 1. 核心求解能力
- 求解随时间任意变化的电流和电压分布
- 支持任意波形的激励源
- 考虑电容效应和电阻效应
- 支持三维模型

### 2. 控制方程
∇·(σ∇V) + ∇·(ε∇(∂V/∂t)) = 0

### 3. 支持的边界条件和激励源
- 与直流电流场完全相同
- 激励源可以是任意时间函数（阶跃、脉冲、正弦、自定义波形）

### 4. 核心后处理计算
- 任意时刻的电位、电场、电流密度分布
- 任意时刻的电流、电压、功率
- 电流和电压的时间变化曲线
- 焦耳热的时间变化曲线
- 总能量损耗计算

### 5. 典型应用场景
- 开关、继电器的瞬态电流分析
- 电容充放电过程分析
- 雷击、浪涌等瞬态过程分析
- 电池充放电过程分析

## 四、高级耦合功能
### 1. 与静电场的耦合
- 开启"Include Insulator Field"选项
- 同时求解导体区域的电流场和绝缘体区域的静电场
- 适用于同时存在电流和静电场的高压设备分析

### 2. 与热场的耦合
- 单向耦合：电流场计算焦耳热，作为热场的热源
- 双向耦合：热场计算温度分布，反馈给电流场更新电导率
- 支持与ANSYS Icepak和ANSYS Mechanical的无缝耦合
- 用于电热联合仿真，如连接器发热分析、电机绕组发热分析

### 3. 与电路的耦合
- 可以将电流场求解器生成的电阻、电容模型导入电路仿真
- 可以将电路仿真的电压、电流波形作为电流场的激励源

## 五、其他实用功能
- **结果导出**：支持导出为VTK、CSV、Tecplot等格式

## 六、对你开发求解器的启示
1. **架构设计**：
   - 设计一个通用的标量电场求解器基类
   - 直流、交流、瞬态电流场求解器都继承自这个基类
   - 共享网格、DOF管理、矩阵装配、线性求解器等基础功能

2. **功能优先级**：
   - 第一优先级：直流电流场（最常用，工业需求最大）
   - 第二优先级：与热场的单向耦合（焦耳热计算）
   - 第三优先级：交流电流场和瞬态电流场

3. **核心后处理**：
   - 重点实现电阻计算、电流计算、焦耳热计算这三个工业最常用的功能
   - 确保这些计算的精度与Maxwell一致（相对误差<1%）


**ANSYS Maxwell中静电场和直流电流场是两个独立的求解器，但它们共享90%以上的底层基础代码**（网格、DOF管理、矩阵装配、线性求解器等），核心差异仅在物理模型、材料参数、激励源和后处理计算这几个特化部分。

## 一、核心区别对比表

| 对比维度 | 静电场(Electrostatic)求解器 | 直流电流场(DC Conduction)求解器 |
|---------|----------------------------|--------------------------------|
| **核心物理模型** | 静止电荷产生的电场，静电平衡状态 | 恒定电流在导体中的流动，稳态导电状态 |
| **控制方程** | 泊松方程：$\nabla \cdot (\epsilon \nabla V) = -\rho$<br>无电荷区退化为拉普拉斯方程 | 电流连续性方程：$\nabla \cdot (\sigma \nabla V) = 0$ |
| **未知量** | 标量电位$V$ | 标量电位$V$ |
| **主要导出量** | 电场强度$\mathbf{E}=-\nabla V$<br>电位移矢量$\mathbf{D}=\epsilon \mathbf{E}$ | 电场强度$\mathbf{E}=-\nabla V$<br>电流密度$\mathbf{J}=\sigma \mathbf{E}$ |
| **关键材料参数** | 介电常数$\epsilon$（相对介电常数） | 电导率$\sigma$（电阻率$\rho=1/\sigma$） |
| **主要激励源** | 电压源（Dirichlet边界）<br>电荷源（体电荷、面电荷） | 电压源（Dirichlet边界）<br>电流源（电流流入/流出边界） |
| **导体处理** | 理想导体内部电场为零，是等势体 | 导体内部有电场和电流，存在电压降 |
| **绝缘体处理** | 是主要求解区域，电场集中在绝缘体中 | 默认不求解绝缘体中的电场<br>（可开启"Include Insulator Field"选项耦合静电场） |
| **核心后处理** | 电容计算、电场力、电能量 | 电阻计算、电流分布、焦耳热、电损耗 |
| **多物理场耦合** | 一般不直接耦合热场 | 可直接与热场耦合（焦耳热→温度→电导率变化） |
| **典型应用场景** | 电容器、高压绝缘子、静电放电、传感器 | 连接器、母线排、接地电阻、电热器、PCB电流分布 |

## 二、关键细节说明

### 1. 为什么它们是两个独立的求解器？
虽然数学形式非常相似（都是二阶椭圆型偏微分方程，都求解标量电位），但它们描述的是完全不同的物理现象：
- 静电场描述的是**电荷与电场**的关系
- 电流场描述的是**电流与电场**的关系

这种差异导致了：
- 单元刚度矩阵的计算不同（一个乘$\epsilon$，一个乘$\sigma$）
- 激励源的类型和施加方式不同
- 后处理计算的物理量完全不同

### 2. 它们之间的耦合关系
Maxwell提供了一个非常重要的功能：**在直流电流场求解器中开启"Include Insulator Field"选项**，这会自动耦合静电场求解器，同时求解：
- 导体区域的电流场（由电导率$\sigma$决定）
- 绝缘体区域的静电场（由介电常数$\epsilon$决定）

这种耦合方式适用于同时存在电流和静电场的场景，例如高压设备的绝缘分析。

## 三、对你开发求解器的启示

你可以完全借鉴Maxwell的架构设计，实现**代码最大化复用**：

### 1. 设计一个通用的标量电场求解器基类
```cpp
class ScalarElectricSolverBase : public PhysicsSolver {
public:
    // 所有标量电场求解器共享的接口
    void initialize() override;
    void assemble() override;
    SolverResult solve() override;

protected:
    // 纯虚函数，由具体求解器实现特化部分
    virtual double getMaterialCoefficient(const Element& element) const = 0;
    virtual void applyExcitations() = 0;
    virtual void computeDerivedQuantities() = 0;

    // 共享的基础功能
    std::shared_ptr<DOFManager> dof_manager_;
    std::shared_ptr<MatrixAssembler> matrix_assembler_;
    std::shared_ptr<LinearSolver> linear_solver_;
    std::vector<double> potential_; // 标量电位解
};
```

### 2. 派生出具体的求解器类
```cpp
// 静电场求解器
class ElectrostaticSolver : public ScalarElectricSolverBase {
protected:
    double getMaterialCoefficient(const Element& element) const override {
        return element.getMaterial().getPermittivity();
    }

    void applyExcitations() override; // 施加电压源和电荷源
    void computeDerivedQuantities() override; // 计算E、D、电容、电场力
};

// 直流电流场求解器
class DCConductionSolver : public ScalarElectricSolverBase {
protected:
    double getMaterialCoefficient(const Element& element) const override {
        return element.getMaterial().getConductivity();
    }

    void applyExcitations() override; // 施加电压源和电流源
    void computeDerivedQuantities() override; // 计算E、J、电阻、焦耳热
};
```

### 3. 实现耦合求解器
如果需要支持"电流场+静电场"耦合，可以再设计一个耦合求解器类，它会同时创建并调用上述两个求解器。

这种架构的优势是：
- 所有基础功能（网格、DOF、矩阵、求解器）只写一次
- 新增电场类求解器（如交流电流场）只需实现特化部分
- 代码维护成本低，bug修复可以在基类中统一进行
