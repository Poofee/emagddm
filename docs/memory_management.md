# 内存管理规范

## 动态内存管理
- 优先使用智能指针（std::shared_ptr、std::unique_ptr）管理动态内存
- 禁止手动使用 new/delete（除底层内存池工具），避免内存泄漏

## 内存预分配
- 迭代过程中尺寸不变的动态数组（如子域解、对偶变量），需通过基础工具层的内存管理模块做一次性预分配
- 禁止迭代中反复申请/释放内存

## 缓存优化
- 高频访问的变量（子域分解因子、对偶变量λ）需做 64 字节缓存行对齐
- 通过基础工具层的内存对齐接口实现

## 资源释放
- 所有类需实现析构函数，释放动态内存、求解器资源、文件句柄
- 主程序结束前，需手动释放全局资源（并行通信、日志对象）
